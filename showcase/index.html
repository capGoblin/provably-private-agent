Architecture: Off-chain Verify + On-chain Attest + x402

[1] Agent (off-chain)
  • • Receives x402 payment (0.1 XLM)
  • • Generates UltraHonk ZK proof
  • • Verifies proof locally via bb
  • • Signs attestation (ed25519)
  • • Submits to executor
[2] Executor (on-chain)
  • • Verifies attestation sig (ed25519)
  • • Stores x402 payment receipt
  • • Stores trade + proof + VK
  • • Emits audit event
[3] Re-verify (any party)
  • • Pulls VK + proof + x402 receipt from chain
  • • Runs bb verify client-side
  • • Math doesn't lie
